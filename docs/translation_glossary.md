# Translation Glossary

Last updated: 2026-02-11

## Canonical Terms

Use these exact terms consistently in UI, notifications, and templates.

- Escrow hold
  - English: `Escrow hold`
  - Spanish: `Cuenta de resguardo`
- Release payment
  - English: `Release payment`
  - Spanish: `Liberar pago`
- Raise issue
  - English: `Raise issue`
  - Spanish: `Reportar inconveniente`
- Joint release instruction
  - English: `Joint release instruction`
  - Spanish: `Instruccion de liberacion conjunta`
- Estimate deposit
  - English: `Estimate deposit`
  - Spanish: `Deposito de estimado`
- Reliability score
  - English: `Reliability score`
  - Spanish: `Puntuacion de confiabilidad`
- High-ticket concierge
  - English: `High-ticket concierge`
  - Spanish: `Concierge de alto valor`

## Tone Guidelines

- Constructive and calm.
- No accusatory language.
- Neutral wording for dispute-related flows.

## Notification Style

- Keep short, action oriented, and neutral.
- Include next action when possible.
- Respect user language preference.
